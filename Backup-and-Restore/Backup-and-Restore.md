# Issue Title: User File Backup and Restore

# Scenario
A user is migrating to a new device and needs important work documents, browser bookmarks, and personal files backed up and restored to a new workstation.

# Environment
- Device: Windows 10/11
- Storage location:
  - External USB drive
- User permission: Standard account

# Backup Tasks Performed
1. Identified folders requiring backup:
   - Desktop
   - Documents
   - Downloads
   - Browser bookmarks
2. Copied files to external storage using File Explorer
3. Verified backup integrity by opening random files
4. Safely ejected storage media

# Restore Tasks Performed
5. Connected external drive to new device
6. Copied files into appropriate user folders
7. Restored browser bookmarks from exported file
8. Confirmed access to user data
9. Cleaned up backups and deleted temp files

# Outcome
User successfully transitioned to a new device with all required files transferred and accessible.

# Skills Demonstrated
- Data protection and backup awareness
- User file structure understanding
- Basic migration workflow
- Validation & post-migration testing

# Best Practices Recommended
- Centralize files in Libraries instead of custom folder locations
- Enable automatic syncing with OneDrive/Google Drive
- Encourage routine data backups

# Tools Used
- Windows File Explorer
- External USB drive or cloud storage
- Browser export/import features
